# NOURVEX — From Vision to Digital Reality

**NOURVEX** (a fusion of "Nour" — light in Arabic, and "Vortex" — energy in motion) is a web development and digital solutions brand by **Nour Mohammed Abdessamed**, a full-stack developer based in El-Baydhe, Algeria. The name represents the convergence of creative vision and technical execution — turning ideas into powerful digital experiences.

![NOURVEX Portfolio Preview](f513c127-8294-42b4-8aa5-4a317468762a.jpeg)

## ✨ Features

- **3D Parallax Hero** — Mouse-tracked depth effect with interactive spotlight mask
- **Animated Typography** — Word-by-word reveal with gradient text effects
- **Interactive Stats Counter** — Animated number counting on scroll
- **Smooth Scroll Reveals** — Elements animate in as you scroll down
- **Floating Navigation** — Fixed header with animated burger menu overlay
- **Service Showcase** — Three-tier pricing cards (Landing Pages, Full Sites, Automations)
- **Client Sign-up Form** — Integrated with Formspree for direct email notifications
- **Playful Micro-interactions** — Button shimmer, logo wiggle, confetti on form submit
- **Fully Responsive** — Optimized for mobile, tablet, and desktop
- **WhatsApp Integration** — Direct chat link for instant client communication

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure & semantics |
| CSS3 | Animations, gradients, grid, responsive layout |
| Vanilla JavaScript | Parallax, scroll effects, form handling, canvas masking |
| Space Grotesk + Inter | Typography |
| Formspree | Form submissions (no backend needed) |
| SVG Icons | Lightweight inline icons |

## 🚀 Getting Started

Simply open `index.html` in any modern browser:

```bash
git clone https://github.com/NourTi/nourvex-portfolio.git
cd nourvex-portfolio
start index.html
```

No build tools, no dependencies.
